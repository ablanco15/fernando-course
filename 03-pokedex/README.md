# Execute in development

1. Clone repository
2. Execute
```
yarn install
```
3. To have Nest CLI Installed

```
npm i -@nestjs/cli
```
4. Run Database

```
docker-compose up -d
```

## Used Stack

* MongoDB
* Nest
